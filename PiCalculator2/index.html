How It Works

This demo works based on the fact that the chance of a random point being inside of a shape is proportional to the shape's area.

To find the circle's area we can use a simple rule.

A = πr2

To find the square's area we use the size length squared.

A = s2

We know that the side of the square is equal to the diameter of the circle. Which is twice the radius.

a = D = 2r
A = a2
= D2
= (2r)2
= 4r2

Finaly we can find the ratio between their areas and rearrange to get π. =D

Acir / Asqu
= πr2 / 4r2
= π / 4
π = 4 * (Acir / Asqu)
